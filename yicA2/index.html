CSS Grid

What I did:

By setting the CSS display property to 'grid', I was able to structure the layout of this webpage using an invisible grid. The main grid is 4 rows and 2 columns across, each of them constituting varying widths to make for a functionally and aesthetically optimal layout. There also exists a nested grid within the middle sector, where the images are positioned. In order to make the webpage 'responsive', I implemented media query properties to re-position the images depending on the dimensions of the viewer's device, or the size of the window.

Flexbox

What I did:

In order to create the 'split' image in the header, I set the display property of the image to 'flex'. Each gap in the image is actually a solid block of color, acting as the flex items in its parent container, while the image serves as a backdrop.
